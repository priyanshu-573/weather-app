# 🌦️ Real-Time Weather Dashboard

A sleek, responsive weather application that provides live weather data and a 24-hour forecast. This project demonstrates advanced Frontend development skills, REST API integration, and modern UI design principles.

## 🚀 Live Demo
**[View Live Project](https://priyanshu-weather-dashboard-55571w.netlify.app)**

## 🛠️ Tech Stack
* **Language:** JavaScript (ES6+)
* **Styling:** CSS3 (Glassmorphism, Flexbox, Grid)
* **API:** [OpenWeatherMap API](https://openweathermap.org/api)
* **Deployment:** Netlify

## ✨ Key Features
* **Live Weather Data:** Fetches real-time temperature, humidity, and wind speed for any city worldwide.
* **24-Hour Forecast:** Displays an hourly breakdown of weather conditions.
* **Interactive UI:** Features a modern "Glassmorphism" design with a blurred card effect.
* **Dynamic Backgrounds:** Background images and icons change automatically based on the weather (Clear, Clouds, Rain, etc.).
* **Mobile-First Design:** Fully responsive layout that looks professional on both mobile and desktop screens.
* **Smart Search:** Supports searching via the "Enter" key for a smoother user experience.
